# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

This repository tracks a food garden in Durham, NC (27707, USDA Zone 7b/8a) with a focus on:
- Planting schedules and timing specific to Durham's climate
- Garden experiments, observations, and lessons learned
- Crop rotation planning and companion planting strategies
- Raised bed and container garden management

## Garden Context

**Location**: Durham, NC 27707 (USDA Hardiness Zone 7b/8a)

**Key Climate Factors**:
- Average last frost: mid-April
- Average first frost: mid-November
- Clay-heavy red soil (for in-ground areas)
- Hot, humid summers
- Mild winters
- Two growing seasons: cool season (fall/winter/early spring) and warm season (late spring/summer/early fall)

**Garden Type**: Raised beds and container gardens

**Priorities**:
- Budget-friendly approaches
- Climate change adaptation
- Organic/sustainable practices
- Learning through experimentation

## Garden Advisor Agent

This repository includes a specialized Claude Code agent (`durham-garden-advisor`) located in `.claude/agents/durham-garden-advisor.md`. This agent should be invoked when users ask questions about:
- What to plant and when in Durham's climate
- Plant selection and variety recommendations
- Troubleshooting plant problems
- Soil preparation and amendments
- Pest management
- Climate adaptation strategies
- Budget-friendly gardening techniques
- Container and raised bed gardening

The agent has deep expertise in Durham-specific conditions and provides practical, actionable advice tailored to this region's unique challenges (clay soil, humidity, heat, etc.).
